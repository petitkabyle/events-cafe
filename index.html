🔥 Connexion Firebase
Les données sont partagées en temps réel entre toi, ton frère et le formulaire client.
📋 Comment connecter (une seule fois) :
1. Crée un projet gratuit sur console.firebase.google.com
2. Active Realtime Database (mode test)
3. Copie tes clés dans le fichier firebase-config.js
4. Recharge la page — le voyant passe au vert ✅
🔔 Notifications par email
Recevez un email sur deux adresses quand un client remplit le formulaire, ou quand un événement est créé / modifié. Via EmailJS (gratuit, 200 emails/mois).
📋 Setup EmailJS (5 min, gratuit) :
1. emailjs.com → Créer un compte gratuit
2. Email Services → Connecter Gmail ou Outlook → copier le Service ID
3. Email Templates → dans le champ "To Email" mets {{to_email}}, puis utilise dans le contenu : {{subject}} {{message}} {{to_name}} {{time}} → copier le Template ID
4. Account → General → copier la Public Key
