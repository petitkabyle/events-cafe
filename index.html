🛠️ Catalogue produits
Ajoutez, modifiez ou désactivez vos produits. Les changements s'appliquent partout : stock, événements, formulaire client.
💡 Astuce : Les produits inactifs n'apparaissent plus dans la fiche de location ni dans le formulaire client, mais l'historique reste intact. Tu peux les réactiver à tout moment.
🔄 Synchronisation automatique : Chaque modification du catalogue (ajout, désactivation, renommage) est instantanément reflétée dans le formulaire client — zéro action manuelle requise.
👥 Clients & Tarifs personnalisés
Enregistrez vos clients avec leurs tarifs sur mesure. Quand ils remplissent le formulaire, le devis est calculé automatiquement avec leurs prix.
🏷️ Tarif de base (par défaut)
Appliqué à tout client inconnu ou sans tarif personnalisé. C'est ta grille « standard ».
💡 Comment ça marche : le système reconnaît le client par le nom de sa société (saisi dans le formulaire). S'il correspond à un client enregistré ici, ses tarifs perso + ses frais de livraison + sa remise sont appliqués automatiquement au devis. Sinon → tarif de base.
🔥 Connexion Firebase
Les données sont partagées en temps réel entre toi, ton frère et le formulaire client.
📋 Comment connecter (une seule fois) :
1. Crée un projet gratuit sur console.firebase.google.com
2. Active Realtime Database (mode test)
3. Copie tes clés dans le fichier firebase-config.js
4. Recharge la page — le voyant passe au vert ✅
📴 Mode hors-ligne
L'application fonctionne même sans connexion (sur les salons, en livraison…). Toutes vos modifications sont enregistrées sur l'appareil et envoyées automatiquement à tout le monde dès le retour d'internet.
🔄 Diagnostic de synchronisation
Vérifie que chaque type de donnée se synchronise bien entre les appareils. Un voyant vert « connecté » ne suffit pas : chaque module a son propre canal, et les règles Firebase doivent autoriser tous les nœuds.
⚠️ Si un nœud est « bloqué » : va dans la console Firebase → Realtime Database → Règles, et mets :
{ "rules": { ".read": true, ".write": true } }
puis Publier. Cela autorise tous les modules (clients, tarifs, livraisons…) d'un coup. Détails dans GUIDE_INSTALLATION.md.
🔔 Notifications par email
Recevez un email sur deux adresses quand un client remplit le formulaire, ou quand un événement est créé / modifié. Via EmailJS (gratuit, 200 emails/mois).
📋 Setup EmailJS (5 min, gratuit) :
1. emailjs.com → Créer un compte gratuit
2. Email Services → Connecter Gmail ou Outlook → copier le Service ID
3. Email Templates → dans le champ "To Email" mets {{to_email}}, puis utilise dans le contenu : {{subject}} {{message}} {{to_name}} {{time}} → copier le Template ID
4. Account → General → copier la Public Key
