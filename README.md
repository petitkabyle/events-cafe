# ☕ Events Café

Système de gestion événementielle (calendrier + stock en temps réel) **et** formulaire client,
synchronisés à plusieurs grâce à Firebase. Thème marron / blanc / noir.

## Fichiers

| Fichier | Rôle |
|---|---|
| `index.html` | L'application de gestion (toi + ton frère) |
| `formulaire.html` | Le formulaire de demande (à partager aux clients) |
| `app.js` / `form.js` | La logique |
| `data.js` | Ton matériel et tes listes (modifiable) |
| `firebase-config.js` | Tes clés Firebase (à remplir) |
| `style.css` | Le thème Events Café |
| `GUIDE_INSTALLATION.md` | **Commence par ici** : le pas-à-pas complet |

## Mise en route

1. Crée un projet Firebase gratuit et colle tes clés dans `firebase-config.js`.
2. Mets le dépôt en ligne (GitHub Pages) ou utilise Netlify Drop.
3. Partage l'URL `…/index.html` avec ton frère et `…/formulaire.html` avec tes clients.

Tout est détaillé dans `GUIDE_INSTALLATION.md`.
